# Library Membership Registration System

## Project Description
The Library Membership Registration System is a web-based application designed for an institutional library.  
It allows users to register as library members by filling out a multi-section form that collects personal, contact, account, and additional information.

The project demonstrates the use of HTML, CSS, JavaScript, and MySQL following proper folder structure and frontend validation techniques.

---

## Features
- Multi-section membership registration form
- 15+ form fields with proper labels and placeholders
- Client-side form validation using JavaScript
- Error and success message handling
- Desktop-first professional UI design
- Organized folder structure
- MySQL database design for storing member data

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- MySQL (XAMPP)
- GitHub

## Validation Implemented
- Empty field validation
- Email format validation
- Password length validation
- Form submission prevention on validation failure

## Database Details
- Database Name: `library_system`
- Table: `members`
- Primary Key: `member_id`
- Stores member personal and membership information
