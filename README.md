# Library Membership Registration System

## Description
A web-based system for registering members in an institutional library.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- MySQL (XAMPP)

## Features
- Multi-section registration form
- Client-side validation
- Responsive desktop-first design
